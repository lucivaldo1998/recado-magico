## Instruções

1. **Gere** cada imagem usando o prompt acima em Leonardo AI (grátis) ou similar
2. **Remova o fundo** se necessário (remove.bg — grátis)
3. **Redimensione** para 512x512px
4. **Salve como PNG** com o nome do slug:
   - `princesa-estrela.png`
   - `fada-luna.png`
   - `sereia-marina.png`
   - `unicornio-iris.png`
   - `super-leo.png`
   - `robo-bip.png`
   - `astro.png`
   - `dino-rex.png`
   - `pirata-jack.png`
   - `ninja-kai.png`
5. **Coloque** em `/public/characters/` (substituindo os SVGs atuais)
6. **Atualize** o componente CharacterAvatar.jsx para usar `.png` em vez de `.svg`
